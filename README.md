## Installing
<div align='center'><p><b>You can install or update <code>python-bale-bot</code> via:</b></p></div>

### PyPi:

```
$ pip install python-bale-bot -U
```

### Git:

```
$ git clone https://github.com/python-bale-bot/python-bale-bot
$ cd python-bale-bot
$ python setup.py install
```
